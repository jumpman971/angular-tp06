Gestion des données : AngularFire FireBase

FIREBASE

FireBase est un produit Google.

FireBase est une base de données en temps réel (Real Time DataBase).

Il y a bien sûr une requête entre le client et le serveur comme une transaction traditionnelle vers du SQL (Rafraichir la page)

Mais une fois que la requête est intialisée, toutes les mises à jour seront accessibles contrairement à une requête traditionnelle SQL

Les MAJ s'afficheront immédiatement sur notre WebSite

Accéder à Firebase

Cliquez sur accéder à la console

  1. Ajouter un projet
  2. Dans le tableau de bord apparaissent tous les projets
  3. Ajouter une application ( IOS - Android - Web ) => Web !
  4. Donner un nom à l'application
  5. Aller dans configuration (Firebase SDK snipet)
  6. Copier la conf dans environnements.ts
  7. La config sera lue depuis le service films.modules grâce au module : AngularFireModule.initializeApp(environment.firebaseConfig)

Configuration des dépendances :

npm install firebase @angular/fire

npm install -g firebase-tools

npm install -g json-server
